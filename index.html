Welcome to the Family Tree Builder

Rules & Instructions

  • Start with the Family Patriarch: Add the eldest family member first.
  • Add Each Member Carefully: Include name, relation, DOB, and description.
  • Upload Images: Optional but recommended for each member. Portraits work best.
  • Select Parent: Every member except Patriarch (male head of a family) must have a parent.
  • Check Details: Click member images to view info in a popup.
  • Tree Layout: The tree automatically adjusts as you add members.
  • Progress: Refreshing the page will reset the tree.
  • Mobile Experience:Rotate your screen for good experience

    • Note: Removing individual family members is not supported. If you want to start over, refresh the page to rebuild the tree from scratch.
